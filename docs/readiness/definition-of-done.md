# Definition Of Done: Stage 0

- ADR set is accepted and committed.
- Domain model, ownership matrix, trust boundaries, and sequence diagrams exist.
- OpenAPI contract validates.
- Workflow schema validates and example fixtures pass semantic checks.
- Backend and frontend skeletons compile against shared packages.
- Draft migrations and RLS test plan exist.
- CI baseline exists for build, lint, test, contract validation, and secret scan.
