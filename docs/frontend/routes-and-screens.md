# Frontend Routes And Screens

## Route Map

- `/dashboard`
- `/library`
- `/automations`
- `/automations/[id]`
- `/automations/[id]/builder`
- `/documents`
- `/chat`
- `/recommendations`
- `/admin/security`
- `/sign-in`

## Screen Intent

- Dashboard: readiness + orchestration overview
- Library: template registry and install state
- Automation detail: workflow preview and required inputs
- Builder gate: embed token contract and blocked state
- Documents: classification and storage state
- Chat: AI gateway readiness, not real provider execution
- Recommendations: advisory inbox only
- Admin/security: secret boundary and release gates
